# Screenshots for the README

Add PNG or JPG files here. The main [README](../../README.md) references these paths.

| File | What to capture |
|------|-----------------|
| `hero.jpg` | Wide shot of the Pi with the round display running radar (good for the top of the README) |
| `assembly.jpg` | Pi + round display hardware (cables mounted, optional case) |
| `radar.png` | Radar screen — aircraft, sweep, map background |
| `flight-detail.png` | Flight detail screen after tapping an aircraft |
| `tracked.png` | Tracked flight screen (route, progress bar, LIVE/ETA stats) |
| `clock.png` | Clock screen with weather |
| `forecast.png` | Multi-day forecast screen |
| `settings.png` | On-device settings screen |
| `web-portal.png` | Web portal in a browser (any section or full page) |
| `stats.png` | Flight statistics page (`/stats`) |

Tips:
- Use the native panel resolution or crop to the visible round area.
- Prefer dark backgrounds to match the app theme.
- Keep filenames lowercase as listed above so README links work without edits.
